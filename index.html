Privacy Policy

Last updated: September 1, 2025

Information We Don't Collect

DRMY is designed with privacy by default. We do not collect, store, or transmit any personal information including:

  • Dream entries or content
  • Personal identifiers
  • Usage analytics
  • Device information
  • Location data

Local Data Storage

All dream data is stored locally on your device using secure, encrypted storage mechanisms provided by iOS. This data never leaves your device unless you explicitly choose to use the optional Google Gemini API integration.

Optional Third-Party Integration

If you choose to use Google Gemini API for enhanced dream analysis:

  • You provide your own API key
  • Dream content is sent directly to Google's servers for analysis
  • We do not store or have access to your API key or the content sent to Google
  • This feature can be disabled at any time
